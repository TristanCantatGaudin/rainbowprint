# rainbowprint

The most basic structure for the package to be installable:

		rainbowprint/
		├── LICENSE
		├── pyproject.toml
		├── README.md
		├── src/
		│   └── rainbowprint/
		│       ├── __init__.py
		│       └── functions.py
		└── tests/

  ## Installation:

  	pip install git+https://github.com/TristanCantatGaudin/rainbowprint.git
